# Тест-кейс №3. Проверка предложения перевода для англоязычных страниц

1. Запустить тестируемое приложение
ОР: Приложение успешно запустилось. Открылся начальный экран приложения со строкой поиска и табло.  
ФР: Соответствует ОР.

2. Тапнуть в поисковую строку и ввести текст «en.wikipedia.org»
ОР: Текст «en.wikipedia.org» введен в строку поиска, отображается кнопка «Перейти».  
ФР: Соответствует ОР.

3. Тапнуть по кнопке «Перейти»
ОР: Открылся новый таб со страницей «en.wikipedia.org», страница отображается на английском, на экране присутствует предложение перевода страницы.  
ФР: Открылся новый таб со страницей «en.wikipedia.org», страница автоматически переведена на русский язык, на экране присутствует панель настройки перевода и поп-ап с предложением перевода надписей на картинках, в адресной строке иконка языка для показа оригинала. Автоперевод стоит по умолчанию.
