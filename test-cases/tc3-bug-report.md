# Тест-кейс №3. Проверка предложения перевода для англоязычных страниц

## Общая информация
| Параметр | Значение |
|----------|----------|
| **Статус тест-кейса** | ⚠️ Частично пройден (значительные расхождения) |
| **Тип проблемы** | Изменение функциональности / Поведенческое расхождение |
| **Вероятная причина** | Обновление логики автоперевода в Яндекс.Браузере |
| **Критичность** | Low (функциональность расширена, не сломана) |

---

## Детализация выполнения

### Шаг 1: Запуск приложения
**Действие:** Запустить тестируемое приложение

**Ожидаемый результат:**
- Приложение успешно запустилось
- Открылся начальный экран приложения со строкой поиска и табло

**Фактический результат:**
- ✅ Приложение запустилось успешно
- ✅ Открылся начальный экран со строкой поиска и табло

**Статус:** ✅ PASS  
**Комментарий:** Стартовый экран отображается корректно.

---

### Шаг 2: Ввод адреса Wikipedia
**Действие:** Тапнуть в поисковую строку и ввести текст «en.wikipedia.org»

**Ожидаемый результат:**
- Текст «en.wikipedia.org» введен в строку поиска
- Отображается кнопка «Перейти»

**Фактический результат:**
- ✅ Текст «en.wikipedia.org» введен в поисковую строку
- ✅ Отображается кнопка «Перейти»

**Статус:** ✅ PASS  
**Комментарий:** Поисковая строка работает корректно.

---

### Шаг 3: Перевод страницы (РАСХОЖДЕНИЕ)
**Действие:** Тапнуть по кнопке «Перейти»

**Ожидаемый результат:**
1. ✅ Открылся новый таб со страницей «en.wikipedia.org»
2. ✅ Страница отображается на английском
3. ✅ На экране присутствует предложение перевода страницы

**Фактический результат:**
1. ✅ Открылся новый таб со страницей «en.wikipedia.org»
2. ❌ **Страница автоматически переведена на русский язык** (вместо английского)
3. ❌ **На экране присутствует панель настройки перевода** (вместо простого предложения)
4. ❌ **Появился поп-ап с предложением перевода надписей на картинках** (новая функция)
5. ✅ В адресной строке появилась иконка языка для показа оригинала
6. ✅ **Автоперевод включен по умолчанию**

**Статус:** ❌ FAIL (изменение поведения)

### Шаг 4: Ожидание 4-5 секунд (РАСХОЖДЕНИЕ)
**Действие:** Подождать 4-5 секунд после загрузки страницы

**Ожидаемый результат:**
1. **Предложение "Перевести страницу на русский"** (из шага 3)
2. **С анимацией свернулось** (плавное уменьшение/преобразование)
3. **В круглый фаб (FAB)** предложения о переводе страницы

**Фактический результат:**
1. ❌ Нет предложения для сворачивания (страница уже переведена)
2. ❌ Нет анимации (UI остается статичным)
3. ❌ Нет круглого фаба (остается полная панель)
4. ✅ Панель управления сохраняется без изменений
5. ✅ Иконка переключения языка в адресной строке

**Статус:** ❌ FAIL (изменение поведения)

### Шаг 5: Взаимодействие с переводом (РАСХОЖДЕНИЕ)
**Действие:** Тапнуть на фаб предложения о переводе

**Ожидаемый результат:**
1. Отображается переведенная на русский язык страница
2. Если появилось предложение перевода надписей на картинках, тапнуть в нем «Перевести»

**Фактический результат:**
1. ❌ Отображается страница на языке оригинала (английский)
2. ❌ Панель настройки перевода скрывается
3. ✅ При нажатии на иконку перевода надписей на картинках, на изображении происходит поиск текста

**Статус:** ❌ FAIL  
**Комментарий:** Функционал был изменен.
